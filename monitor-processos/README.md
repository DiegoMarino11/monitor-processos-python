# Monitor de Processos em Python

Script que verifica se programas (como Chrome, Python, etc.) estão rodando no seu computador.

## Como Usar
1. **Instale as dependências** (só na primeira vez):
   ```bash
   pip install -r requirements.txt

2. Execute o script:
   ```bash
   python monitor.py
